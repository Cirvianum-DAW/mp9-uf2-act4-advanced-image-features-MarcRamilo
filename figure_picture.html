Exercise 7: Figure Elements

Using <figure> and <figcaption> Elements

The <figure> element is used to encapsulate media, such as images, diagrams, illustrations, etc., along with their captions using the <figcaption> element. It helps to associate the caption with the media content, making it more accessible and understandable to users.

Code

            <figure>
                <img src="/images/ex1.avif" alt="Example Image">
                <figcaption>This is an example image.</figcaption>
            </figure>

Exercise 8: Art Direction - Picture elements

Using <picture> Element for Art Direction

The <picture> element allows us to provide different image sources based on media conditions. This helps in optimizing images for different screen sizes and resolutions, improving performance and user experience.

Code

            <figure>
                <picture>
                    <source srcset="/images/ex1.avif" media="(min-width: 1200px)">
                    <source srcset="/images/ex1_768.png" media="(min-width: 768px)">
                    <source srcset="/images/ex1_767.png" media="(max-width: 750px)">
                    <img src="/images/ex1.avif" alt="Example Image">
                </picture>
                <figcaption>This is an example image with art direction.</figcaption>
            </figure>
