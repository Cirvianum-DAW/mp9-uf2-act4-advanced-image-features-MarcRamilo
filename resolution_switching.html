Exercise 3: Resolution Switching - Modern Img Tags

Using the <img> element with srcset and sizes attributes:

Code: 
            <img srcset="/images/ex1.avif 400w,
            /images/ex1.avif 800w,
            /images/ex1.avif 1200w" sizes="(max-width: 600px) 400px,
                      (max-width: 1200px) 800px,
                      1200px" src="/images/ex1.avif" alt="An example image with different resolutions">

The srcset attribute provides multiple sources for the browser to choose from, each with different resolutions. The sizes attribute specifies the sizes of the image based on media conditions. This allows the browser to choose the most appropriate image source based on the device's screen width and resolution.
