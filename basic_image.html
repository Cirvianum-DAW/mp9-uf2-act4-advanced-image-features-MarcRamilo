Exercise 1: Basic Image Insertion

    <img src="/images/ex1.avif" alt="1.jpg">

The alt attribute is important for images because it provides alternative information for an image if a user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader). The alt attribute should describe the content of the image, and it is also used by search engines to index the image and understand the content of the image.

Exercise 2: Image Formats and Attributes

Using different image formats and attributes:

Example 1: WEBP Format

Using an image in WEBP format:

            <img src="/images/ex2_webp.webp" alt="A stunning image in WEBP format" width="518" height="690">

Example 2: AVIF Format

Using an image in AVIF format:

            <img src="/images/ex2_avif.avif" alt="A breathtaking image in AVIF format" width="563" height="375">

The difference between the real size of the image and the dimensions specified in the width and height attributes affects how the image is displayed on the webpage. The specified dimensions can either stretch or compress the image, altering its aspect ratio and visual appearance. It's important to maintain the original aspect ratio to prevent distortion.
